HOW TO VIEW CONTENTS OF IELEX DATA
-----------------------------------

1. Once downloaded, please keep all of these files together in the same directory/folder. Should the files that make up the contents of IELEX be separated the hypertext links WILL NOT WORK
2. Double click on 'HEADPAGE' to open the hypertext landing page in your web browser
3. Navigate between datasets on this page by using the links provided on the landing page
4. To view 'PUBLICATIONS' (publications that have made use of this data) and 'BIBLIOG' (Bibliopgraphic information) use the same method of opening as per Step 1.

Note: On both Mac and Windows Operating System, if you do not have HTML files set to open as default in a web browser you may need to right click and select 'Open With' before selecting an available web browser.
